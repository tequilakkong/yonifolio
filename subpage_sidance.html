OVERVIEW

메인 페이지에서 무용 축제와 관련된 주요 소식들을 한 눈에 찾기 어려운 문제를 ‘퀵 메뉴'와 ‘공지사항' 등 핵심 정보를 메인에 배치하여 개선하였습니다. 또한 메뉴 개수가 간소함에도 서브 메뉴 내 동일 콘텐츠가 중복되어 필요한 정보를 찾는데 시간이 걸리는 문제를 파악하여 메뉴 구조를 개선하였습니다.

CONCEPT

현장의 생생함을 더하기 위해 메인 배너에 공연 사진을 추가해 몰입감을 높였습니다. 다양한 공연을 한 데 배치하되 주의가 산만해지지 않도록 메인 색상을 공통적으로 활용하여 통일감을 주었습니다. 전체적으 미니멀한 디자인에 볼드한 서체와 애니메이션을 추가하여 단순하지만 생생한 느낌을 주었습니다.

중복 스타일의 class 적용
같은 스타일이 반복 사용되는 것을 방지하고자 레이아웃이나 폰트 등 자주 사용되는 스타일의 클래스를 생성하여 적용하였습니다. 이 방법은 html 태그에 바로 사용할 수 있어 간편하다는 장점이 있었지만 디자인을 수정해야 할 경우 일일이 태그로 찾아가 수정을 해야하는 문제를 발견할 수 있었습니다.

CSS 동적 애니메이션 구현
동적인 애니메이션을 구현하는 과정에서 transition과 @keyframes를 사용하였습니다.

float을 이용한 요소 배치
텍스트와 이미지 등의 배치는 모두 float 속성을 사용하였으며, float이 적용된 요소의 크기만큼 차지하던 영역이 사라지는 문제를 해결하는 방법에 대해 배웠습니다. 여기서는 가상요소 after를 만들어 clear 속성을 부여하는 방법과 부모 요소에 높이를 지정해주는 방법을 사용하였습니다.
